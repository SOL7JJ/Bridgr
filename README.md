# Bridgr

Standalone HTML prototypes for the Bridgr product experience.

## What is in this repo

- `index.html`: local launcher for the prototype pages
- `bridgr-landing page.html`: marketing landing page
- `bridgr-brand-planner.html`: brand planning flow
- `bridgr-brand-planner_updated.html`: revised brand planning flow
- `bridgr-creator-planner.html`: creator planning flow
- `bridgr-creator-editor.html`: creator profile editor
- `bridgr-creator-profile.html`: creator profile page
- `bridgr-product-profile-editor.html`: product profile editor
- `bridgr-product-profile.html`: product profile page

## Run locally

Open `index.html` directly in your browser.

On macOS:

```bash
open index.html
```

You can also open any individual `.html` file directly.

## Notes

- These pages are static HTML prototypes.
- External font dependencies were removed so the project works locally without requiring font downloads.
